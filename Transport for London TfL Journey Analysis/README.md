> A professional SQL analytics project demonstrating data engineering,
> analytical SQL, and business intelligence skills using Transport for
> London passenger journey data.

------------------------------------------------------------------------

## 📌 Project Overview

This project explores historical Transport for London (TfL) passenger
journey data using **Snowflake SQL** to answer real-world business
questions that support transportation planning, operational
decision-making, and trend analysis.

The notebook demonstrates how SQL can transform raw transportation data
into meaningful business insights through efficient querying,
aggregation, filtering, ranking, and reporting.

This project is part of my Data Engineering portfolio and reflects
industry-standard SQL practices used by Data Engineers, Analytics
Engineers, and Business Intelligence professionals.

------------------------------------------------------------------------

## 🎯 Business Objectives

-   Identify the transport modes with the highest passenger volumes.
-   Evaluate the popularity of the Emirates Airline service over time.
-   Determine the years with the lowest Underground & DLR usage.
-   Demonstrate analytical SQL techniques for business reporting.

## 📊 Dataset

The project uses the **Transport for London (TfL) Journey Dataset**,
loaded into the Snowflake table:

``` sql
TFL.JOURNEYS
```

The dataset contains historical passenger journey information across
multiple London transport services.

# Project Tasks

## 1. Most Popular Transport Modes

Ranks transport types by total passenger journeys using aggregation.

## 2. Emirates Airline Popularity Analysis

Returns the highest recorded passenger volumes for the Emirates Airline
service.

## 3. Least Popular Underground & DLR Years

Identifies years with the lowest Underground & DLR passenger journeys.
